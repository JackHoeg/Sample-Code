README
This Backgammon Tournament Manager was made by Jack Hoeg and Ben McCormick for the
Software Construction course at Northwestern University.

Info regarding the requirements and progression of this project can be found here:
https://users.cs.northwestern.edu/~robby/courses/393-2021-spring/

Info on the installation of Rust and Cargo is accessible here: 
https://doc.rust-lang.org/cargo/getting-started/installation.html

To run the unit and integration tests, go to the root directory and call 'cargo test'
from your CLI.

Other projects can be seen at jackohoeg.com/code
